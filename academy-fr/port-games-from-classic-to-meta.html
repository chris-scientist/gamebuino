Convertir des jeux de la Gamebuino Classique à la META

La Gamebuino classique a de nombreux bons jeux. Bonne nouvelle : C'est super simple de les convertir afin de les utiliser sur votre Gamebuino META !

Débutant

Académie
Convertir des jeux de la Gamebuino Classique à la META

Installer les librairies

Tout d'abord, commencez par suivre le tutorial Débuter avec votre Gamebuino. Vous n'avez pas besoin de la librairie de la Gamebuino classique mais vous aurez à installer 2 librairies complémentaires afin de pouvoir compiler les jeux de cette console pour les utiliser sur votre META.

  • https://github.com/Gamebuino/Gamebuino-Compat
  • https://github.com/Gamebuino/Gamebuino-EEPROM

Téléchargez le fichier .zip pour ces 2 fichiers puis dans l'éditeur Arduino, cliquez sur "Croquis/ Inclure une bibliothèque / Ajouter la bibliothèque .ZIP...".

Modifier la version 'classique' du jeu

Maintenant, vous pouvez ouvrir le code source d'un jeu de la Gamebuino Classique et effectuer les modifications ci-dessous:

Au début du code, vous devriez trouver quelque chose ressemblant à ça: ( Si besoin, utilisez CTRL+F pour faire une recherche).

Remplacez

#include <Gamebuino.h>
Gamebuino gb;

Par

#include <Gamebuino-Compat.h>

Remplacez (s'il est présent)

#include <EEPROM.h>

Par

#include <Gamebuino-EEPROM.h>

Retirez (s'il est present)

#include <SPI.h>

Compiler le jeu pour la META

  • Cliquez sur "Outils/Cartes/Gamebuino META" (en bas de la liste) pour selectionner le bon matériel.
  • Cliquez sur "Croquis/Téléverser" ou faites un "Ctrl + U" pour compiler et transférer le programme dans la Gamebuino META.

Corriger les erreurs

Si vous avez quelques messages d'erreur, pas d'affolement: le code nécessite quelques ajustements. Remontez pour trouver le premier de la liste et essayez de le corriger puis relancez la compilation. Recommencez cette étape jusqu'à ce que vous n'ayez plus d'erreur. Si vous n'arrivez pas à en résoudre une, lisez les messages des autres personnes dans les commentaires. Une personne a peut être déjà trouvé une solution, sinon ajoutez votre question dans les commentaires afin que nous puissions vous aider :)

Partager le résultat

Maintenant que vous avez réussi, vous devriez Partager une création. Ainsi, les autres utilisateurs pourrons jouer directement avec votre jeu sans avoir eux-même à le convertir et vous laisseront de délicieux pouces en l'air :)
