Affichez du texte à l'écran

Étape 2
Étape terminée ?

Maintenant que vous avez lancé votre premier programme, il est temps de mettre un peu les mains dans le cambouis !

Revenons sur le code que nous avons exécuté à l’étape précédente.

from gamebuino_meta import begin, waitForUpdate, display, buttons, color

while True:
    waitForUpdate()
    display.clear()

    display.print("Hello world")

La notion principale à connaitre est qu’un programme en Python est composé d’instructions. Chaque instruction équivaut à une ligne de code, et les instructions sont exécutées les unes à la suite des autres. display.print("Hello world") est donc une instruction permettant d'écrire Hello world à l'écran ! Ne vous occupez pas des autres instructions pour le moment.

Challenge #1

Essayez de modifier le code pour écrire à l’écran la phrase « Je m’appelle Bob ». Si vous observez bien le code, cela ne devrait pas trop être compliqué !

Besoin d'aide ?

  • L'instruction display.print("xxx") permet d'afficher du texte à l'écran
  • Les guillemets autour de votre texte sont très importantes, ne les oubliez pas !
  • N’oubliez pas de sauvegarder votre code afin de l’exécuter sur votre Gamebuino à chaque fois que vous faites des modifications

Note : si après avoir sauvegardé votre code rien ne se passe sur l'écran de votre Gamebuino, qui reste figé, c'est sans doute qu'il y a une erreur ! Dans ce cas, cliquez sur le bouton "Série" (celui avec les deux flèches) dans le menu de Mu. Cela va ouvrir une petite fenêtre en bas, dans laquelle vous aurez quelques indications en cas d'erreur, et surtout la ligne concernée ! N'hésitez pas à garder cette fenêtre ouverte tout au long de ce workshop, elle va surement vous aider plus d'une fois !

Résultat attendu

from gamebuino_meta import begin, waitForUpdate, display, buttons, color

while True:

Explications

Ce n’était pas très difficile, n’est-ce pas ? Il suffisait juste de remplacer « Hello world » par « Je m’appelle Bob ».
