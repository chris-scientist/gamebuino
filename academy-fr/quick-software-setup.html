Installation rapide du logiciel

Étape 1

Installation de l'IDE Arduino; logiciel vous permettant de taper votre code et de l'envoyer sur la Gamebuino via le câble USB.

Nous allons installer l'IDE Arduino qui vous permettra de taper votre code et l'envoyer sur la Gamebuino par le câble USB.

Certains d'entre vous ont peut-être déjà l'IDE Arduino installé sur leur ordinateur. Si vous êtes concernés, il vous reste à installer manuellement les outils nécessaires pour la Gamebuino.

Pour ceux qui partent de rien, passez directement aux étapes ci-dessous pour une installation rapide !

Téléchargez l'Arduino IDE portable pré-configuré

Vous avez de la chance, nous vous avons préparé une version pré-configurée et prête à l'usage. Téléchargez celle qui correspond à votre système d'exploitation et extrayez l'archive :

  • Windows
  • Linux

Attention, si vous êtes utilisateur Mac, l'installation automatique n'est plus possible. Nous vous invitons à suivre ce tuto à la place : configuration manuelle d'Arduino.

Ensuite, décompressez le fichier que vous venez de télécharger à l'instant.

Cela peut être un peu long alors n'hésitez pas a faire un tour dans les créations et a mettre quelques +1 sur ce qui vous plait !

Une fois la décompression terminée, ouvrez le dossier et lancez "arduino".

Et voilà, le logiciel est prêt : vous pouvez passer à la suite !
